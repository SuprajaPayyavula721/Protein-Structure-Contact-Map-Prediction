## 📌 Overview
This project tackles the highly complex problem of protein structure prediction using **Apache Spark** for Big Data processing (originally developed for the ECBDL'14 Self-deployment track). The goal is to accurately predict residue-residue contacts within proteins, which is a crucial step in understanding protein folding mechanisms and biological functions.

## 🚀 Key Features
- **Massive Scale:** Analyzed an incredibly large and imbalanced dataset comprising **32 million protein instances**, each with **631 features**.
- **Distributed Computing:** Leveraged Apache Spark's Resilient Distributed Datasets (RDD) and parallel processing capabilities to handle the massive data load efficiently.
- **Advanced Methodologies:** 
  - Implemented Divide and Conquer strategies.
  - Utilized Ensemble Machine Learning methods to improve prediction accuracy on the highly imbalanced dataset.

## 🛠️ Tech Stack
- **Platform:** Apache Spark (PySpark)
- **Domain:** Bioinformatics, Big Data Analytics
